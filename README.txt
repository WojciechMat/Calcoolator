Podstawowe informacje:
Program potrzebuje pliku wejściowego którego ścieżkę (lub nazwę jeśli jest w tym samym folderze co exe) należy podać jako argument przy uruchamianiu programu. 
Tworzy (lub modyfikuje) plik "out.txt" który jest uzupełnionym o wyniki plikiem wejściowym."
Możliwe operacje to dodawanie, mnożenie, dzielenie, modulo, zamiana podstawy systemu liczbowego i potęgowanie. 
Maksymalna długość luczby to 80 cyfr (zarówno dla wejścia jak i wyniku).
